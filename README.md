# subject

현재 완료된 사항
1. 회원가입 및 로그인
2. 입금 및 송금
3. 최근 5회간의 송수금 기록 조회
4. 로그인 한 사용자에게 토큰을 발급하기
5. 사용자의 등급에 따라 접속 권한 부여
6. url 입력으로 접근하기 차단

현재 미완료 상황
1. 차단 버튼을 구현했으나, 아직 기능은 구현하지 못함
2. 접근을 차단하지만, 알림을 구현하지 못함

느낀 점
1. 프로그램이 무엇을 요구하는지 잘 모르겠다
   -> 어떤 기능을 만들기 위해서 무엇부터 가져와야 할 지 잘 모르겠다. Controller -> Service -> Repository의 순서로 프로세스가 구성되는 것은
      이해하지만 그 사이에 무엇을 가지고 오거나, 어떤 내장되어 있는 메소드를 사용하거나 하는 것들을 무엇을 써야할지 잘 모르겠다.
   
2. 문법을 외우기가 어렵다
   -> 요구하는 문법이나 활용도는 무궁무진한데, 사실 아는게 잘 없으니 응용도, 어떤 개념을 활용하거나 깊이 있는 이해를 갖추기 어려운 것 같다

3. 사실상 복습하는 시간이나 마찬가지인 것 같다. 이전에 배운게 새롭게 느껴지고 몇 개는 기억이 나기도 하지만, 내가 아는게 정확한지 확신이 없다.
